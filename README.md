# Restaurant Website

## Project Overview
A polished, beginner-friendly restaurant landing page built with HTML5 and vanilla CSS. The project showcases a warm, modern design with clear sections for the hero area, menu, gallery, testimonials, contact information, and footer.

## Technologies Used
- HTML5
- Vanilla CSS

## Features
- Responsive header and navigation
- Hero section with call-to-action button
- About section with image and story
- Menu cards with food images and prices
- CSS-only gallery carousel
- Testimonials section with guest reviews
- Contact section with address, hours, and form
- Footer with quick links and social placeholders

## Concepts Practiced
- Semantic HTML
- HTML page structure
- Flexbox
- Responsive layouts
- Cards
- Forms
- Navigation
- Images
- CSS styling
- Media queries
- Typography
- Spacing
- Hover effects

## Folder Structure
- index.html
- style.css
- README.md

## What I Learned
This project helped reinforce the basics of building a complete landing page using semantic HTML and CSS. It also strengthened understanding of layout techniques like Flexbox, spacing, image handling, form structure, and responsive design.
